对于终端输入的时候总会遇到一长串命令不方便修改; 命令记不全等问题， 以下内容可能让你感觉极度舒适>>>
<!-- more -->
## 移动
```sh
ctrl a    # 移动至命令开始
ctrl e    # 移动至命令结束
```

## 删除
```sh
ctrl l    # 清屏 和clear的功能一致
ctrl u    # 删除光标以前的所有字符
ctrl k    # 删除光标以后的所有字符
ctrl w    # 删除光标以前的一个单词
```
## 补全
```sh
tab       # 自动补全
tab tab   # 连续两下tab 查看以输入字符开始的所有命令
```
